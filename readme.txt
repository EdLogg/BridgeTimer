This program is an app that will show the remaining time for
the movement in a bridge game.  The expectation is the video output
can be displayed on a large TV or other device for everyone to see.

The program allows you to set the number of rounds, the number of 
board per round, the number of minutes per board, and the amount time 
between movements.  Once the session is started the number of rounds 
and the number of boards per round will be locked.  However, you can 
change the number of minutes per board at any time.  There are also 
buttons to increase or decrease the timer by 30 seconds if needed.

The program allows you to set the amount of time when no new boards
should be started.  There is an audio alarm for this as well as
an alarm when the round is over.  These sounds can be replaced by 
replacing the existing .wav files.

The screen will show the remaining time to play or the time to move
in large numbers.  The round number is shown as well as the expected
end time for this game.  If the round is finished before the time 
runs out just press the End Round button.  If for some reason you 
want to pause play select the Pause Timer button.  Press Resume
Timer when you are ready to comntinue.
